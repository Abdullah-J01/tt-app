# App icons

Add the following PNGs here (referenced by `public/manifest.webmanifest`):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512 (used as maskable)

Also add `apple-touch-icon.png` (180×180) at `public/` for iOS.
Generate all sizes from one source SVG with a tool like `@capacitor/assets` or `pwa-asset-generator`.
